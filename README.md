# Sociedade União Sintrense — Jekyll (GitHub Pages)

## Publicar
1. Cria um repositório no GitHub e faz upload destes ficheiros.
2. Settings → Pages → Deploy from a branch → `main` / `(root)`.

## Trocar tema rapidamente
No `_config.yml`, muda a linha:

- `theme: minima`

para um dos temas suportados pelo GitHub Pages, por exemplo:
- `theme: jekyll-theme-cayman`
- `theme: jekyll-theme-slate`
- `theme: jekyll-theme-midnight`
- `theme: jekyll-theme-minimal`
- `theme: jekyll-theme-architect`

Depois faz commit/push.
